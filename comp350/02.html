In this session we are going to look at the build process on the PS4 Dev kit using Unreal and Unity, before finally looking at native builds. We are also going to examine how a Profiler can be used to check performance of your code.

Learning Objectives:

By the end of this session, you should be able to:

  • Develop games for the PS4
  • Understand the usage of a profiler
  • Profile your own code base
