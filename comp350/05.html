This session will introduce various experimental design and statistical analysis techniques. These will be demonstrated using R, a free software package for statistical computing. Statistical analysis of empirical data is an essential part of almost every research project, and so knowledge of the appropriate tools and techniques is vital to answering your research question.

Learning Objectives:

By the end of this session, you should be able to:

  • Explain key concepts in experimental design and statistical analysis
  • Use R to analyse empirical data
