When servers serve data, how do games interpret and react to the response? This session answers this question. With a huge growth in Internet-enabled games and social media sharing systems, a working knowledge of such data packaging, parsing, deserialisation, and client-server testing approaches is key for computing professionals in the games industry.


Studio Practice

Each week you will engage in mandatory studio practice under the supervision of mentors who will help you to develop your collaborative programming practice. You are also expected to attend a mandatory meeting with your product owner each week.


Workshop
Learning Objectives:
  • Explain how to retrieve data from a remote server within a game
  • Parse mark-ups including XML and JSON
  • Incorporate remote data into a live game development project
Agenda:

0050 - 0135 Workshop - Libraries and Approaches to Parsing XML and JSON
