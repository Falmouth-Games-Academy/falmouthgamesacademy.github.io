This session introduces the concept of version control and related practices including following conventions and refactoring. You will use git and GitHub to leverage version control in your programming projects. This is important because skills in version control are essential for all computing professionals. Employers will expect you to work in a team and will only consider hiring you if you have mastered the practical use of version control.
