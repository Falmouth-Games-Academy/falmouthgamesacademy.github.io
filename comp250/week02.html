This session will cover some techniques for designing intelligent behaviours, using simple rule-based systems and look-ahead search. These techniques are the foundation upon which more complex AI systems are built.

Learning Objectives:

By the end of this session, you should be able to:

  • Design simple rule-based AI systems
  • Explain key theoretical concepts behind decision making
  • Use game theory to reason about the behaviour of intelligent agents
