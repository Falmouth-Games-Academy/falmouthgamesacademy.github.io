This session refreshes your knowledge of finite state machines, and introduces behaviour trees. These tools are commonly used in video games to allow developers to design complex and believable character behaviours without hard-coded rules.

Learning Objectives:

By the end of this session, you should be able to:

  • Explain how finite state machines and behaviour trees are used in AI
  • Design character behaviours using behaviour trees
  • Implement an AI system based on behaviour trees
