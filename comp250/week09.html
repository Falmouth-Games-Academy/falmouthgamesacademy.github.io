This session will introduce the concept of machine learning, particularly with neural networks. Machine learning is a powerful technique for creating AI systems which can adapt to new problems, going beyond the knowledge the designer puts into the system. Recent advances by Google DeepMind and others have shown the promise of these techniques.

Learning Objectives:

By the end of this session, you should be able to:

  • Explain the operation of machine learning and neural networks
  • Use deep learning to tackle challenging AI problems
  • Implement Skynet, bringing about the destruction of civilisation as we know it
