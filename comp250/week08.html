This session introduces some common architectures for procedural content generation (PCG) in games. PCG was originally developed as a method for fitting more content into less storage space, however in recent years it has gained in popularity as a way for smaller teams to produce larger games, and to allow styles of game which would be difficult or impossible to achieve with hand-authored content.

Learning Objectives:

By the end of this session, you should be able to:

  • Design PCG systems for games
  • Explain the operation of an evolutionary algorithm
  • Implement an evolutionary PCG system
