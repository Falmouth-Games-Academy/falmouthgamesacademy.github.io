This session covers pathfinding and navigation in games. Many genres of game require AI agents to navigate complex 2D and 3D environments. We will revisit the A* pathfinding algorithm, and see how it can be applied in practice in video games.

Learning Objectives:

By the end of this session, you should be able to:

  • Implement the A* pathfinding algorithm
  • Explain the application of navigation meshes in 3D games
  • Design AI systems which can navigate complex 3D terrain
