This session introduces Monte Carlo Tree Search (MCTS). MCTS is a search-based AI approach based on random simulation, which has proven extremely successful in many complex board games and other domains, despite its relative simplicity. Only a few commercial video games have used MCTS to date, however it is growing in popularity.

Learning Objectives:

By the end of this session, you should be able to:

  • Explain the MCTS algorithm
  • Implement an MCTS-based AI system
  • Explain some common enhancements to MCTS
