This session introduces the concept of an algorithm, and some commonly used notations for communicating algorithms. Programming is not just about writing code; in this session you will learn some other ways of expressing computational ideas.

Learning Objectives:
  • Recall some basic algorithms
  • Produce and explain basic flowcharts
  • Produce and explain basic pseudocode
