This session introduces the way in which data is represented in computer memory. A fundamental understanding of basic data types, and their limitations, is important in understanding the behaviour of your programs.

Learning Objectives:
  • Recall how basic data types are represented in binary form in memory
  • Explain the notion of objects and use them to model systems
  • Write your first object oriented program
