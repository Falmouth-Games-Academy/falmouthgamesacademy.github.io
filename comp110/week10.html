In this session you will begin to transfer your knowledge of Python programming to C++. Programming in C++ is more challenging than Python, but also more powerful and more widely used in the games industry.

Learning Objectives:
  • Recall the key differences between Python and C++
  • Use the Visual Studio IDE to compile and run your first C++ program
  • Write basic procedural and object-oriented programs in C++
