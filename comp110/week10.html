In this session you will learn about how computer processors execute programs, and the various ways in which the programs you write can be translated and executed by the computer. This will give you a better understanding of how your programs will behave when they execute.

Learning Objectives:
  • Explain the difference between interpretation, just-in-time compilation and ahead-of-time compilation
  • Describe how common high-level code structures translate to machine code
  • Explain how integer and floating-point numbers are represented in the computer
