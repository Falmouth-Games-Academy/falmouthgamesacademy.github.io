In this session you will learn about the various ways in which the programs you write can be translated and executed by the computer. This will give you a better understanding of how your programs will behave when they execute.

Learning Objectives:
  • Explain the difference between interpretation, just-in-time compilation and ahead-of-time compilation
  • Distinguish the basic parts of a compilation pipeline and recall how they operate
  • Describe how common high-level code structures translate to machine code
