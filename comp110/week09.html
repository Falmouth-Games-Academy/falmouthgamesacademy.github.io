This session explores how basic data types such as booleans, numbers and strings, as well as complex data types such as objects, are represented in the computer.

Learning Objectives:
  • Explain the representation of common "plain old data" types in memory
  • Distinguish pass-by-reference and pass-by-value
  • Distinguish allocation of memory on the stack and on the heap
