This session is the second of three to explore concurrency in depth. You will explore issues associated with the structure of concurrent applications, taking specific note of task execution and threading pools. This will provide further insight into the design of the architecture for the server for the educational MUD.


Learning Objectives:
  • Analyse the structure of a concurrent application
  • Identify code smells in concurrent code
  • Explain threading pools
