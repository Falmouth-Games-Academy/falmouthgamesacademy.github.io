This session sets the stage for the rest of the module: a practice-orientated series where you will develop an educational multiplayer game that will teach young people basic computer programming. The first half of this session involves the exploration of MUDs (multi-user dungeons) and MMORPGs (Massively Multiplayer Online Roleplaying Games) and invites you to design of an educational MUD. The second half outlines some basic concepts for constructing client-server multiplayer games using the World of Zuul as inspiration.


Learning Objectives:
  • Describe the history of MUDs and MMORPGss
  • Design an educational MUD
  • Implement a basic text-based adventure game in Java
