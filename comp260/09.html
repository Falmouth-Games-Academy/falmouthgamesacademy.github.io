Lecture:

This session will continue are exploration of cloud native concepts and techniques. We will begin with a short introduction to Kubernetes, followed by a technical demo on how to create a very basic Kubernetes cluster using Digital Ocean instances as nodes.

Learning Objectives:
  • Outline the components of a Kubernetes Cluster
  • Define the terms deployment, pods and services in the context of Kubernetes
  • Orchestrate a basic three node cluster
Steps to Complete the Session:
  • Watch the presentation about Kubernetes and Helm
  • Explore the Kubernetes official interactive tutorials (link)
  • Watch the Kubernetes Cluster Demo video
  • Replicate the Kubernetes Cluster setup from the demo video.
