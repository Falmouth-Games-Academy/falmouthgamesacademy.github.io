This session introduces the processing pipeline used by modern high-performance GPUs, and the OpenGL API which we will be using to interact with this pipeline. Modern GPUs operate quite differently to CPUs, and understanding these differences is essential to making full use of the GPU's power.

Learning Objectives:
  • Recall the key stages of the graphics pipeline
  • Explain the differences between a CPU and a GPU
  • Write basic programs using SDL and OpenGL
