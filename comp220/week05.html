This session introduces how to render complex meshes in a 3D environment and how to map textures onto them. These are the fundamental techniques of 3D graphics that almost all 3D games make use of.

Learning Objectives:
  • Explain how a complex 3D model is represented in memory
  • Explain how a 2D texture image can be wrapped onto a 3D model
  • Write programs which draw textured meshes to the screen
