This session introduces Interleaved Vertex Data and Element Buffers. We will examine how these improve performance of 3D Applications. You will also gain an understanding of how Vertex Data is stored in memory

Learning Objectives:
  • Explain how a complex 3D model is represented in memory
  • Use an element buffer to imporve rendering performance
