This session introduces the agile philosophy as well as practical guidelines surrounding game development practice as they apply to software. This is important because you are studying digital games and many of the key principles and processes are drawn from the software software engineering literature. Once you develop your programming ability, you will need to apply the agile philosophy in order to efficiently create digital games.


Workshop
Learning Objectives:
  • Explain the agile philosophy and how it is applied in SCRUM
  • Assess the key attributes and qualities of user stories
  • Identify appropriate user stories based on a game design
