This session introduces the important foundational concepts concerning: games as artifacts; game vocabulary and terminology; models of game experiences; formal game elements; game development practice; commerce; and communication skills. These will inform your idea of what game design is as well as inform your development practices.


Morning Session
Learning Objectives:
  • Explain the difference between the terms: game; game concept; and game design
  • Recognize the formal elements of games and the MDA model
  • Design a simple board game
  • Compare and Contrast approaches to game development

Afternoon Session
Learning Objectives:
  • Explain how to pitch a game concept
  • Compare the different audiences that you pitch to
  • Extend existing game designs
  • Communicate a game design through a 30-second 'elevator' pitch
