This session introduces more advanced techniques in software project management and helps you to explore the application of agile development practices in a practical way. Using branching, for example, is a key approach to distributing work to peers to be completed without causing disruption to the master version of the software. Such knowledge is essential in professional contexts and will demand considerable effort on your part to fully develop. As such, we introduce these techniques now to provide you sufficient time to practice ahead of your next game project, which you are now preparing with your peers from other routes.


Workshop
Learning Objectives:
  • Use branching to improve your game development workflow
  • Design appropriate unit tests to evaluate the functionality of code excerpts
  • Apply continuous integration techniques to your game development workflow using TravisCI
