This session will investigate how JavaScript and Canvas can be leveraged to write games and game engines. We will walk through the construction of several popular game genres and build JavaScript versions.

Learning Objectives:

By the end of this session, you should be able to:

  • Create technical architecture for creating JavaScript games
  • Apply software engineering to appropriately implement games in JavaScript
  • Evaluate the strengths and weaknesses of design patter choices
