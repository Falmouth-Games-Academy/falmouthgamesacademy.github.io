This session will investigate how JavaScript and Canvas can be leveraged to applications and technical demonstrators. We will walk through the construction of diverse application and build them in JavaScript.

Learning Objectives:

By the end of this session, you should be able to:

  • Create technical architecture for creating JavaScript applications
  • Apply software engineering to appropriately implement applications in JavaScript
  • Evaluate the strengths and weaknesses of design patter choices
