This session will introduce JavaScript as an interpretted object-oriented programming language for creating vertical web-hosted applications.

We will assess JavaScript's suitability for game, application and demo writing against the standard input-process-output model.

Learning Objectives:

By the end of this session, you should be able to:

  • Explain the JavaScript object model in relation to ECMAScript version
  • Apply suitable design patterns to implementing functionality in JavaScript
  • Evaluate the strengths and weaknesses of JavaScript and HTML5 for creating particular game genres and design patterns
