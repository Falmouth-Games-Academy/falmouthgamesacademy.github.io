This session will introduce technical portfolios as a technique for job hunting and the role and scope of hosted JavaScript as an approach for reducing the barriers to recruiters seeing your work

Learning Objectives:

By the end of this session, you should be able to:

  • Explain the benefits of creating technical portfolios as a job hunting tool
  • Recall how to set up a suitable development environment for creating JavaScript applications
  • Use a Digital Oceans serverlet to host JavaScript powered applications
