This session introduces the overall architecture of a game engine. Games of different genres often have some commonality in their architecture, so understanding the common features of game engine architecture will be useful in all of your game development projects.

Learning Objectives:
  • Explain the main components of a game engine
  • Distinguish the phases of a game's main loop
  • Use standard game engine architectures in your own projects
