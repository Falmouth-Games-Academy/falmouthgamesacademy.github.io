This session introduces features of the SDL library that allow your program to load graphical assets in file formats such as JPEG and PNG, and to render text using font files.

Learning Objectives:
  • Use the SDL library to write complex programs
  • Use SDL extension libraries in your programs
  • Write programs that load graphics and fonts from external files
