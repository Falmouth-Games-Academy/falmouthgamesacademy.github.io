This session discusses the fundamentals of game engine design. Nowadays very few game developers implement their own game engines from scratch. However an understanding of the principles of game engine design will help you understand why general-purpose engines such as Unity and Unreal are structured the way they are, and allow you to adapt and troubleshoot the engine more easily.

Learning Objectives:

By the end of this session, you should be able to:

  • Explain the main components of a game engine
  • Apply this understanding to existing engines such as Unity and Unreal
  • Implement your own game engine (should the need ever arise)
