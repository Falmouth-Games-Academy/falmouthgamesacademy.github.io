Many games do not store data on the device hosting a game, but rather communicate data to the device from a repository on the Internet. Typically, most games are, or interact with, web applications. As such, a working knowledge of web architecture and server technologies is essential for all technical professionals in the games industry.


Studio Practice

Each week you will engage in mandatory studio practice under the supervision of mentors who will help you to develop your collaborative programming practice. You are also expected to attend a mandatory meeting with your product owner each week.


Workshop
Learning Objectives:
  • Explain the key features and architectures of web applications
  • Setup a remote server using ssh/telnet and UNIX commands
  • Deploy a web database that will generate XML/JSON output
