This session introduces Simple DirectMedia Layer (SDL), an open-source library for creating 2D games in C++. You will be using this library extensively in your projects this semester and in future. We will also look at the Google C++ conventions, a set of coding style guidelines that will help you produce consistently readable code.

Learning Objectives:
  • Recall the basic components of the SDL library
  • Write basic programs that interact with SDL
  • Write programs that conform to the Google C++ conventions
