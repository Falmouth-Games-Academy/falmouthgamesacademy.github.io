This session introduces the key principles of digital images. Such images are used extensively in games and so it is important to appreciate the various formats and how to manipulate them.

Learning Objectives:
  • Explain how pictures are digitised into raster images by a computer system
  • Apply knowledge of colour models to write programs that manipulate colour in a raster image
  • Use functions, arguments, and basic data structures such as matrices to write programs that manipulate raster images
