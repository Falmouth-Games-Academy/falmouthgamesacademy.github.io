This session introduces more complex manipulations of images using more sophisticated conditions and conditional statements. Exploring these different algorithms reveal different and interest ways to be creative with images.

Learning Objectives:
  • Explain how conditional logic can manipulate the output of a computer program
  • Apply mathematical knowledge to write computer programs that manipulate raster images
  • Trace an existing computer program to predict its output for a given input
