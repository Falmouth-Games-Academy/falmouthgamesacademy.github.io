This session introduces more sophisticated image manipulation techniques using a variety of programming constructs, such as nested loops and conditions. This session will equip you with the skills to finalize your interactive mood board.

Learning Objectives:
  • Apply iteration and nested iteration to write a program that manipulates part of a raster image
  • Copy an image from one array to another
