In this session, you will review the programs of two peers. You are expected to provide insight into the creative aspects of the work as well as evaluate the quality of the source code. This is an important opportunity to receive feedback on your programming practice and to improve your program prior to submission.

Learning Objectives:
  • Identify the key aspects of maintainable code
  • Apply linters to source files
  • Conduct a code review
  • Describe how code could be improved
Agenda:

0100 - 0200 PASS Session - Formatting a Submission
0200 - 0300 Workshop - Tidying and Documenting Code
0300 - 0400 Activity - Peer-Reivew of Code
