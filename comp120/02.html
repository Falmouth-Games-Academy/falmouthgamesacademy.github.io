In this session you will be introduced to Integrated Development Environment (IDE) software, and begin to write and understand simple programs in the Python programming language.

Learning Objectives:
  • Explain the role and basic functions of the IDE
  • Produce functional code in Python
  • Apply pair programming practices to solve a problem
