This session explores the role of computing professionals in the games industry. It will prompt you to consider what type of job you want when you graduate. Further to this, you will be introduced to key learning theories associated with programming education as well as successful learning practices, such as pair programming.

Learning Objectives:
  • Analyse the role of computing professionals in the games industry
  • Recall important theories about learning computer programming
  • Apply pair programming practices to solve a problem
