This session introduces you to basic tone generation and the relevant key theories associated with this topic. You will then use this knowledge to construct a primitive synthesizer. This is important because many games generate sounds that musical instruments cannot produce in order to evoke greater intrigue and a wider range of emotions.

Learning Objectives:
  • Explain key theories associated with digital sound waves
  • Write a program that will generate a complex tone using a composite of sine functions
  • Write a program that will manipulate audio
